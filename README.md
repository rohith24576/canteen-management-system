# Amrita Canteen Management System

A high-performance, interactive web application designed to optimize the campus dining experience. This system provides a unified platform for students and faculty to access menus across multiple campus canteens, customize bakery orders, track nutritional wellness through mood-based recommendations, and manage digital transactions seamlessly.

## Core Capabilities

### Integrated Authentication Framework

- **Secure Access Control**: Standardized Login and Sign-up interfaces with optimized transitions.
- **Security Intelligence**: Real-time validation of password complexity and strength.
- **Account Recovery**: Automated password reset workflow.
- **Advanced UI Design**: Dynamic background rendering with geometric abstraction and modern typography.

### Intelligent Dashboard and Wellness Tracking

- **User Personalization**: Context-aware welcome protocols for registered users.
- **Mood-Based Recommendation Engine**: Proprietary logic suggesting nutritional choices based on emotional states (e.g., Happy, Sad, Tired, Stressed).
- **Behavioral Analytics**: Visual data tracking of user mood trends over a weekly period.

### Strategic Multi-Canteen Integration

- **Main Canteen**: Centralized dining hub specializing in traditional daily menus.
- **MBA Canteen**: Designated social and professional networking dining venue.
- **IT Samudra**: Specialized facility focusing on diverse culinary offerings.
- **Central Navigation Hub**: A consolidated landing page (`intepage.html`) for efficient routing between dining facilities.

### Bakery Customization Engine

- **Technical Design Interface**: End-to-end custom cake procurement system.
- **Configurable Parameters**: Selection of weight, flavor profile (Vanilla, Chocolate, Strawberry, Butterscotch), frosting specifications, and multiple topping options.
- **Personalization Protocols**: Support for custom messaging and scheduled delivery fulfillment.
- **Dynamic Valuation**: Real-time pricing calculations based on selected technical specifications.

### Transaction and Cart Management

- **Streamlined Procurement**: Multi-category cart functionality for efficient ordering.
- **Inventory Control**: Direct adjustment of quantities via dish interfaces or centralized cart management.
- **Preference Synchronization**: Bookmark functionality for frequently accessed menu items.
- **Notification System**: Real-time status updates regarding order fulfillment and promotional offerings.

### Financial and Billing Infrastructure

- **Multi-Channel Payment Integration**: Support for Credit/Debit Cards, UPI, Digital Wallets, and Cash on Delivery.
- **Digital Invoicing**: Automated generation of itemized receipts including service fee calculations.
- **Documentation Export**: Native browser support for direct printing and digital export of invoices.
- **Transaction Archives**: Comprehensive historical logging of all past orders and payment statuses.

### Modern Interface Architecture

- **Theme Management**: Native support for Light and Dark modes with persistent user preference storage.
- **Responsive Layouts**: Fully optimized for cross-platform compatibility, ensuring consistent operation from mobile to desktop environments.
- **Interaction Design**: Implementation of smooth transitions, state-aware animations, and interactive hover states.

## Technical Specification

- **Structure**: HTML5 (Semantic and ARIA compliant).
- **Presentation**: CSS3 (Modular vanilla CSS, glassmorphism effects, and custom animation keyframes).
- **Logic**: JavaScript (ES6+ State Management and DOM Manipulation API).
- **Typography**: Poppins (Precision-weighted modern typeface).
